Financial Transaction Dashboard (Excel)
Overview

This project is an interactive Excel dashboard designed to monitor and analyze e-commerce performance. It tracks sales, orders, vendor details, and inventory, providing actionable insights for better decision-making.

Features

Sales Analysis: Daily and weekly sales trends, top-selling products, and slow periods.

Inventory Tracking: Monitors stock levels and alerts for restocking.

Order Management: Analyzes orders from multiple channels (Swiggy, Zomato, etc.).

Automation: Uses Excel formulas, conditional formatting, and pivot tables to streamline reporting.

Interactive Visuals: Charts and tables for easy data interpretation.

Tools / Technology

Microsoft Excel (Formulas, Pivot Tables, Charts)

Data cleaning and processing

How to Use

Open the Excel file.

Update the raw data sheet with your own sales and inventory data.

Explore the dashboard to view insights, charts, and reports.

Benefits

Quickly identifies sales trends and top-performing products.

Helps in efficient inventory management.

Reduces manual reporting time with automated calculations.

Future Improvements

Integrate with live sales channels for real-time data updates.

Add advanced Excel macros for more automation
